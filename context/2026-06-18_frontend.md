# 2026-06-18 Frontend: просмотр новостей

## Стек

- React 19 + TypeScript + Vite
- **styled-components** — стили в `<Component>.styles.ts` рядом с компонентом
- nginx в Docker — статика + proxy `/api` → `api:8080`

## Структура стилей

```
src/
├── app/
│   ├── app.tsx
│   └── app.styles.ts
├── components/
│   ├── source-select/
│   │   ├── source-select.tsx
│   │   └── source-select.styles.ts
│   ├── news-list/
│   ├── news-detail/
│   └── pagination/
├── hooks/
│   ├── use-news/use-news.ts
│   └── use-sources/use-sources.ts
├── api/client.ts, types.ts
└── styles/theme.ts, global.styles.ts
```

Паттерн: папка `name-of-component/` + файлы `name-of-component.<ext>` в kebab-case.

## Запуск

```bash
docker compose up -d --build
# UI: http://localhost:5181
```

Локально (нужен API на :8080):

```bash
cd frontend && npm run dev
# http://localhost:5181
```

## Функционал

- Выбор источника
- Список новостей с пагинацией
- Панель деталей с полным текстом (`content`)
