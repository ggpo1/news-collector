# Генератор «второго дня»

## API
`POST /api/news/{id}/second-day-angles` — по запросу (как AI rewrite), таймаут `AiRewrite` (30 мин).

## Что собирается из БД
1. **Исторические параллели** — архивные новости (старше 30 дней) с пересечением сущностей или той же категории, исключая SameTopic-связи.
2. **Покрытие события** — другие источники по SameTopic link (для сверки цифр).
3. **Сущности** — из `news_entity_mentions` + co-mention соседи.

## Ollama
Один JSON-ответ: параллели, stakeholders, противоречия в цифрах, 5–7 `suggestedAngles`.

Использует основной `ollama` API (тяжёлая модель).

## UI
Кнопка **«Углы на 2-й день»** в карточке новости → панель с темами, параллелями, игроками, фактчек-углами.

## Зависимости
- `news-entity-extractor` — сущности
- `topic-linker` — SameTopic для противоречий
- Архив новостей старше 30 дней — для параллелей
