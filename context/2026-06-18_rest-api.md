# 2026-06-18 REST API модуль

## Проект

`NewsCollector.Api` — ASP.NET Core Web API на порту **8080**.

## Эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/health` | Health check (БД) |
| GET | `/api/sources` | Список источников |
| GET | `/api/sources/{id}` | Источник по ID |
| GET | `/api/news?page&pageSize&sourceId&hasContent` | Новости с пагинацией |
| GET | `/api/news/{id}` | Детали новости (с `content`) |
| GET | `/api/news/{id}/related` | Связанные новости |

## Docker

```bash
docker compose up -d --build
# API: http://localhost:8080
# Swagger (Development): http://localhost:8080/swagger
```

`RunMigrations=false` для API (миграции делает worker).

## Локально

```bash
dotnet run --project src/NewsCollector.Api
```

## CORS

`Cors:AllowedOrigins` в appsettings — пустой массив = разрешены все origins (для dev).
